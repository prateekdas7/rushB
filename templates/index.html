About RushB
System Features:
  • Create and manage CS:GO tournaments with round-robin format
  • Track teams, players, coaches, and substitutes
  • Manage fixtures with multiple phases (Scheduled, Map Selection, In Progress, End)
  • Record player statistics (Kills, Deaths, Assists, KDA)
  • Automatic leaderboard calculation with win/draw/loss points
  • Manage maps, officials, and servers
Workflow:
  1. Create a tournament with basic information
  2. Add teams with players and coaches
  3. Set up maps, officials, and servers
  4. Start the tournament to generate fixtures
  5. Update fixtures as matches progress
  6. View real-time leaderboard standings
